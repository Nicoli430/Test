# Rails simple blog app

## Api only


```
  Ruby 3.0.+
  Rails 7
```

#### Clone the repo into your local machine

* Install Gems

```
  $ bundle install
```

* Migrate the database

```
  $ rake db:migrate
```

* Seed the database

```
  $ rake db:seed
```

* Run the server

```
  rails s
```

* Open a new terminal tab and test the apis
